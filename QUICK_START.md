## 🎮 如何使用

### 文件操作
```
1. 在左侧文件树中点击文件夹展开
2. 点击文件名在编辑器中打开
3. 文件会在新标签页中打开
4. 点击标签上的 X 关闭文件
```

### 终端命令
```bash
help      # 显示所有可用命令
clear     # 清空终端
echo 文本  # 输出文本
ls        # 列出文件
pwd       # 当前目录
date      # 显示日期时间
whoami    # 显示用户
```

### 面板调整
```
1. 将鼠标移到面板分隔线上
2. 分隔线会变蓝色
3. 拖动调整面板大小
4. 释放鼠标完成调整
```

### 快捷操作
```
- 点击活动栏图标切换面板显示
- 顶部菜单栏提供各种选项
- 状态栏显示当前文件信息
```

## 🎨 自定义主题

编辑 `app/globals.css` 中的 CSS 变量来自定义主题：

```css
:root {
  --background: 240 10% 3.9%;    /* 背景色 */
  --primary: 217 91% 60%;         /* 主色 */
  --accent: 283 65% 58%;          /* 强调色 */
  /* 更多变量... */
}
```
