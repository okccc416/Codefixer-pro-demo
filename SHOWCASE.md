## 🎯 核心功能演示

### 1. 文件资源管理器 📁
```
点击操作:
  📂 文件夹 → 展开/折叠
  📄 文件 → 在编辑器中打开

视觉效果:
  选中: 蓝灰色高亮
  悬停: 浅灰色背景
  图标: 彩色文件类型标识
```

### 2. Monaco 编辑器 📝
```
功能特性:
  ✨ 语法高亮 (Python, JS, TS, MD)
  ✨ 代码折叠
  ✨ 迷你地图
  ✨ 行号显示
  ✨ 多标签页
  ✨ IntelliSense

快捷键:
  Ctrl+F: 查找
  Ctrl+H: 替换
  Ctrl+/: 注释
  Alt+↑↓: 移动行
```

### 3. 集成终端 💻
```
支持命令:
  help    → 显示帮助
  clear   → 清空终端
  echo    → 输出文本
  ls      → 列出文件
  pwd     → 当前目录
  date    → 日期时间
  whoami  → 当前用户

配色:
  背景: 深黑 #09090b
  光标: 蓝色闪烁
  文本: 浅色 #e4e4e7
  命令: 彩色输出
```

### 4. AI 助手 💬
```
消息类型:
  👤 用户消息 (右侧，蓝色)
  🤖 AI 消息 (左侧，灰色)

交互:
  Enter → 发送
  Shift+Enter → 换行
  自动滚动到最新消息
```

### 5. 可调整面板 🔧
```
水平调整:
  文件资源管理器 ↔ 编辑器 ↔ AI 助手

垂直调整:
  编辑器 ↕ 终端

手柄效果:
  默认: 灰色
  悬停: 蓝色高亮
  拖动: 实时预览
```


### 核心组件库
```

```
